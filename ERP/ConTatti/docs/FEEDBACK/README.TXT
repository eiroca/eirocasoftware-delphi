   Sono cambiati i nomi dei programmi, ora la versione a 16bit si chiama 
   ConTat16.exe, mentre la versione a 32bit e' stata rinominata in ConTatti.exe


:) Ai vecchi utilizzatori di "Con Tatti" versioni precedenti alla 0.97.

   Bisogna eseguire la procedura sotto elencata per aggiornare il database.
   
    Eseguire le seguenti operazioni:

    1) Eseguire una copia del database (sottodirectory DB della directory di 
       Con Tatti)
    2) Accertarsi di avere eseguito correttamente il punto 1)
    3) Eseguire DBPatch
    4) Eseguire tutte le patch con il bottone abilitato
    5) Se non ci sono stati errori passare al punto 9)
    6) Attenzione eseguire il punto 8 comporata la distruzione dei dati delle 
       comunicazioni
    7) Se non si vuole eseguire il punto 8) compattare il DB e mandarmelo x la 
       revisione
    8) cancellare tutti i file comunic*.* nella directory DB
    9) Lanciare ConTatti
   10) Eseguire SUBITO la compattazione del database (menu contatti voce 
       Compattazione database)
